# KabyMangadexDL
卡比的mangadex下载器
# MangadexDL
```
KabyMangadexDl/
main_app.py < 运行这个，主GUI（下载队列 + 卷本构建 + 设置）-
```
## 安装（虚拟环境，装在G盘不占C盘）
把整个文件夹放到 `指定目录` 之后手动敲命令：
```
cd 指定目录
```
python -m venv venv
```
venv\Scripts\activate
```
pip install --cache-dir .\pipcache -r requirements.txt
```
之后运行程序：
venv\Scripts\python.exe main_app.py
## 用法
### 1. 下载队列（解决"只能下一部漫画"的问题）
- 点 **"＋ 新建下载任务"**：弹窗里贴网址/搜标题 → 选语言 → 载入章节列表 →
  勾选要下载的章节（默认全选）→ 选输出目录/是否打包cbz → "添加到队列"
- 想同时下好几部漫画，重复点"＋"加多个任务就行，它们会按"设置"里的并发数一起跑
- 点 **"开始下载队列"** 才会真正开始下载（先加任务，攒好了一起启动也行）
- 每个任务的状态/进度在列表里实时更新，选中某个任务点"取消选中任务"能中途停掉它
### 2. 卷本构建（这里是重点，手动+自动两种都能用，不冲突）
- **"扫描章节文件夹"**：选一个目录（比如下载队列输出的那个漫画文件夹，
  里面是一个个 `Ch.xx` 子文件夹），自动识别所有章节
- 左边是"未分配章节"列表，右边是"卷列表"（卷底下挂着已分配的章节）
- **手动分卷**：左边多选几话（Ctrl/Shift点选）→ 右边选中目标卷 → 点"加入→选中卷"；
  想拿出来就在右边选中某几话 → 点"←移出所选卷"，变回未分配
- **自动分卷**：点"自动分组未分配章节"，输入每卷几话，会把**当前未分配**的章节
  自动打包成新的卷，已经手动分配好的卷不受影响——所以你可以先手动挑几个特殊的卷
  （比如番外合集），剩下的话再让自动分组打包，两种方式混着用
- 卷可以"新建空卷"（自己起名字）、"重命名选中卷"、"删除选中卷"（删除后章节变回未分配，不会丢文件）
- 导出：选格式（cbz/epub/pdf）、选输出目录，"导出选中卷"或"导出全部卷"
### 3. 设置
默认输出目录、下载并发数、默认语言、默认打包选项、自动分卷默认每卷话数、默认导出格式，
保存后写入 `config.json`，下次打开程序自动带入（新建任务/新建分组时的默认值，不影响已有任务）。
## 关于并发下载
`设置` 里的"并发数"控制同时跑几个下载任务的线程，MangaDex 官方 API 对请求频率有限制，
建议 2~3，别开太高，不然容易被限速（429），反而更慢。
## PDF 导出的小提醒

PDF 是把所有图片原样拼起来的多页文件，体积会比 cbz 大不少（因为没有专门为漫画优化的压缩/翻页），
适合想用普通 PDF 阅读器/打印的场景；日常在漫画阅读器里看，cbz 还是更轻量的选择。
